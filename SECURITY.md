# SECURITY POLICY

## Supported Versions

We provide security updates for the following versions of UniZ:

| Version | Supported |
| :------ | :-------- |
| v1.x    | Yes       |
| v0.x    | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in UniZ, please report it to us as soon as possible. We take security seriously and will investigate all reports.

Please do not open a public issue for security vulnerabilities. Instead, contact the project maintainers directly.

We will acknowledge your report and keep you informed of our progress in resolving the issue. We appreciate your help in keeping UniZ secure.
