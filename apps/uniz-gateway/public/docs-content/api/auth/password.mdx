---
title: "Password"
description: "Reset a forgotten password using an OTP reset token, or change a password while logged in."
---

UniZ provides two password management endpoints:

- **Reset** — for users who have forgotten their password. Requires a `resetToken` obtained through the [OTP verification](/api/auth/otp) flow. No active session needed.
- **Change** — for logged-in users who want to update their current password. Requires a valid Bearer token.

<Info>All new passwords must be at least **6 characters** long.</Info>

---

## Reset password

`POST /auth/password/reset`

Sets a new password using the `resetToken` issued by `POST /auth/otp/verify`. This endpoint does not require an active session.

**Auth required:** No

### Request body

<ParamField body="username" type="string" required>
  The username whose password is being reset.
</ParamField>

<ParamField body="resetToken" type="string" required>
  The short-lived token returned by `POST /auth/otp/verify`. Tokens are
  single-use and expire shortly after they are issued.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password. Minimum 6 characters.
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  `true` when the password was updated successfully.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable confirmation.
</ResponseField>

### Example

```bash
curl --request POST \
  --url https://api.uniz.rguktong.in/api/v1/auth/password/reset \
  --header 'Content-Type: application/json' \
  --data '{
    "username": "O210008",
    "resetToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "newPassword": "newSecurePassword123"
  }'
```

**200 OK**

```json
{
  "success": true,
  "message": "Password reset successfully"
}
```

### Error responses

| Status                  | Meaning                                                                                                       |
| ----------------------- | ------------------------------------------------------------------------------------------------------------- |
| `400 Bad Request`       | `resetToken` is invalid, expired, or already used. New password does not meet the minimum length requirement. |
| `404 Not Found`         | No account found for the given username.                                                                      |
| `429 Too Many Requests` | Rate limit exceeded.                                                                                          |

## Change password

`POST /auth/password/change`

Updates the password for the currently authenticated user. The caller must supply their existing password to confirm the change.

**Auth required:** Yes — `Authorization: Bearer <token>`

### Request body

<ParamField body="currentPassword" type="string" required>
  The user's current password.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The desired new password. Minimum 6 characters.
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  `true` when the password was changed successfully.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable confirmation.
</ResponseField>

### Example

```bash
curl --request POST \
  --url https://api.uniz.rguktong.in/api/v1/auth/password/change \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "currentPassword": "oldPassword123",
    "newPassword": "newSecurePassword456"
  }'
```

**200 OK**

```json
{
  "success": true,
  "message": "Password changed successfully"
}
```

### Error responses

| Status             | Meaning                                                                                    |
| ------------------ | ------------------------------------------------------------------------------------------ |
| `400 Bad Request`  | `currentPassword` is incorrect. New password does not meet the minimum length requirement. |
| `401 Unauthorized` | Missing or invalid Bearer token.                                                           |
